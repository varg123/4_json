# Читабельный JSON

Cкрипт выводит содержимое `json` в удобном для чтения виде.

# Быстрый старт

Для запуска указывается путь до файла данных в формате `json`
Пример запуска скрипта на Linux, Python 3.5:

```bash

$ python pprint_json.py -f <путь до файла>
```
